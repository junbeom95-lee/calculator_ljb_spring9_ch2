# Calculator_ljb_spring9_ch2
* spring9기 ch2 계산기 과제

1. Java 기초 문법을 이용한 간단한 계산기 프로젝트
2. 사용자로부터 입력 받아 사칙연산을 수행하고 출력합니다.
---
# 기술 스택
- Java 17
---
# 주요 기능
* 덧셈, 뺄셈, 곱셈, 나눗셈 기능
* 실행 중 결과값 저장 기능
